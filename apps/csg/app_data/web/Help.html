Help

The Course Site Generator is a desktop Java application for creating and editing course website details. The app presents 5 tabs, one that lets you edit the course webpage settings, one that lets you edit the syllabus details, one that lets you adjust the meeting times of the course, one that lets you adjust the office hours of the TAs, and one that lets you edit the schedule of the course. The application saves data to a JSON format that can be used by Web pages for displaying course websites.

Step 1: Edit Site Details

In this tab, you can customize the course website details

Step 2: Edit Syllabus Details

Add relevant info to the course syllabus section

Step 3: Edit Meeting Times Details

Add the times that your course will meet for either lectures, recitations, or labs

Step 4: Edit TA Office Hours

Add or Edit the times that the TAs for the course will hold their office hours

Step 5: Edit Schedule Details

Add relevant event details for homeworks, holidays, and more!
